# Protect It Undercoating Website

A professional website for **Protect It Undercoating**, a mobile rust prevention service serving the Tristate Area.

## Quick Start

Simply open `index.html` in your web browser to view the website locally.

```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

## Project Structure

```
protect-it-undercoating/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## Features

- **Responsive Design** - Looks great on desktop, tablet, and mobile
- **Mobile Navigation** - Hamburger menu for smaller screens
- **Smooth Scrolling** - Animated scroll to sections
- **FAQ Accordion** - Expandable FAQ section
- **Contact Form** - Opens email client with pre-filled message
- **Scroll Animations** - Elements fade in as you scroll
- **Dark Industrial Theme** - Bold, rugged aesthetic fitting for rust protection

## Deployment

This is a static website with no build process required. To deploy:

1. **GitHub Pages:** Push to a GitHub repository and enable GitHub Pages
2. **Netlify:** Drag and drop the folder to Netlify
3. **Vercel:** Connect your repository or drag and drop
4. **Traditional Hosting:** Upload files via FTP to your web host

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --color-rust: #c45c26;        /* Primary accent */
    --color-rust-light: #e07830;  /* Light accent */
    --color-charcoal: #1a1a1a;    /* Background */
}
```

### Content
Edit `index.html` to update:
- Service descriptions
- FAQ questions and answers
- Service area towns
- Contact information
